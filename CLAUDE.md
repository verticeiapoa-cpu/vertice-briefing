# Vértice Briefing

## Idioma
Sempre responda em português brasileiro. Seja direto e objetivo.

## Sobre o projeto
Formulário de briefing em 6 etapas para onboarding de clientes da Vértice Agência Digital.
Coleta todas as informações do cliente e gera automaticamente um prompt para criar o site no Lovable.

- **URL:** https://verticeiapoa-cpu.github.io/vertice-briefing/
- **Repositório:** github.com/verticeiapoa-cpu/vertice-briefing
- **Tecnologia:** HTML, CSS, JavaScript puro (sem frameworks)

## Fluxo das 6 etapas
1. **Escolha do pacote** — Landing Page, Site Completo, Site + Google, Agenda Online, etc.
2. **Sobre o negócio** — nome, responsável, segmento, cidade, endereço, horário, descrição
3. **Serviços** — lista com nome, descrição curta e preço
4. **Identidade visual** — paleta de cores e estilo visual
5. **Contato e redes sociais** — WhatsApp, Instagram, Facebook, TikTok, portfólio
6. **Detalhes finais** — funcionalidades extras, prazo preferido, observações

## Saída
Ao finalizar, gera um **prompt completo** para colar no Lovable e criar o site automaticamente.
